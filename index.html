URL shortener API

A microservice to return a shortened URL that redirects to the query URL.

To use, direct your browser to https://shorturl-marshallaf.herokuapp.com/new/, followed by the query URL. The response will be a JSON object containing the original URL and the shortened version. Visit the shortened URL to be redirected to the original URL.

example usage
// create new URL
https://shorturl-marshallaf.herokuapp.com/new/http://imgur.com/gallery/zWeJ5

// response
// visiting shortUrl will redirect the browser to originalUrl
{
shortUrl: https://shorturl-marshallaf.herokuapp.com/6,
originalUrl: http://imgur.com/gallery/zWeJ5
}
